Installation for SpeakScents API

Backend Installation
1. Install node js from https://nodejs.org/en/
2. Check if installed by running in terminal 
    node -v
3. In terminal install these dependence by injecting this line:
    node install express body-parser multer connect-flash tedious ejs
   If you get an error, you may need to install npm, node install 
4. Check to see you have the dependencies isntalled by typing in terminal, 
    npm list
